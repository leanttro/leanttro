FeirasDeRua

Case de Sucesso | Guia de Feiras de Rua de SP

O "Feiras de Rua" é um produto digital completo que lista feiras em São Paulo. Mais do que um site, é um ecossistema de dados projetado para provar competências de ponta a ponta em Engenharia de IA.

A aplicação inclui um chatbot de IA que utiliza RAG (Retrieval-Augmented Generation): ele busca dados ao vivo do banco PostgreSQL e os injeta na API do Gemini para fornecer respostas precisas e atualizadas sobre as feiras cadastrada.

  • Backend API (Python/Flask)
  • Banco de Dados (PostgreSQL)
  • IA - RAG (Gemini API)
  • Analytics (Power BI)
  • Admin (Jupyter Notebook)
  • Hospedagem (Render/Cloud)
